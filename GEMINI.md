# Directory Overview

This repository is a personal knowledge base and collection of notes focused on various cybersecurity topics. It serves as a study guide and a reference for concepts, tools, and techniques across different domains of cybersecurity. The content is primarily in Markdown format and is organized into folders based on specific subjects, certifications, or platforms.

# Directory Structure and Key Topics

The repository is organized into several directories, each corresponding to a specific area of study within cybersecurity. Here are some of the key topics covered:

*   **`Advent of Cyber 2022/`**: Notes and files related to the Advent of Cyber 2022 challenge by TryHackMe.
*   **`API Penetration Testing/`**: Notes on testing and securing APIs.
*   **`AWS Solutions Architect/`**: Study notes for the AWS Solutions Architect certification.
*   **`Azure/`**: Notes related to Microsoft Azure certifications (AZ-900, SC-200, SC-900).
*   **`Bug Bounty/`**: Information and techniques for bug bounty hunting, including Google Dorking and notes on the OWASP Top 10.
*   **`Computer Networking/`**: Notes on the fundamentals of computer networking.
*   **`eJPT/`**: Materials for the eLearnSecurity Junior Penetration Tester certification.
*   **`HackTheBox/`**: Notes and write-ups for challenges on the HackTheBox platform.
*   **`Linux/`**: Study notes on Linux, based on books like "Linux Basics for Hackers" and "How Linux Works".
*   **`PortSwigger Academy/`**: Notes from the PortSwigger Web Security Academy, covering topics like SQL Injection.
*   **`Security+/`**: Study materials for the CompTIA Security+ certification.
*   **`Splunk/`**: Notes on using Splunk for security monitoring and data analysis.
*   **`TCM Security/`**: Notes from courses by TCM Security, such as Practical Ethical Hacking (PEH).
*   **`TryHackMe/`**: Notes and write-ups for rooms and challenges on the TryHackMe platform.

# Usage

This repository is intended for personal study and reference. The notes are organized for easy access to specific topics. Users can browse the directories to find information on a particular subject. The notes can be viewed with any Markdown reader. The repository can be cloned for offline access:

```bash
git clone https://github.com/tarunsaiks/Cyber-Notes-Repo.git
```
